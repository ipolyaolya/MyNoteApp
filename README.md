# NotesApp

NotesApp - список, с помощью которого можно управлять заметками:
- добавлять
- сортировать по дате создания
- редактировать
- обновление даты при редактировании
- удалять
- расшаривание
- подключена CoreData 

> **ТЗ в EVO**
